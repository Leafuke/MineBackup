# Code of Conduct

## Our Commitment

We want MineBackup to be a welcoming, safe, and respectful project for everyone.
Contributors, users, and maintainers are expected to treat each other with respect
in all project spaces.

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub issues, pull requests, and code review discussions
- Release discussions and documentation feedback
- Community spaces directly related to MineBackup

## Expected Behavior

- Be respectful, patient, and constructive
- Assume good intent and give actionable feedback
- Focus discussion on technical issues and project goals
- Accept responsibility for mistakes and learn from them

## Unacceptable Behavior

- Harassment, threats, hate speech, or personal attacks
- Discrimination based on personal characteristics
- Trolling, insulting language, or deliberately disruptive behavior
- Publishing private information without permission
- Repeated off-topic or bad-faith interactions

## Enforcement

Project maintainers are responsible for interpreting and enforcing this policy.
They may remove comments, lock threads, reject contributions, or apply temporary
or permanent bans for behavior that violates this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior:

1. Prefer contacting the repository owner directly through the GitHub profile:
   https://github.com/Leafuke
2. If a private channel is not available, open an issue with minimal details and
   ask maintainers to move follow-up to a private conversation.

Please avoid posting sensitive personal information in public reports.
